## 실습일지

### 새로 배운 내용
* shell    
사용자의 명령을 해석하여 커널에 전달한다. linux 운영체제의 기본 shell은 bash이다.
* 작업 예약
주기적으로 매일 해야 하는 작업을 예약할 때는 cron을 사용하고, 일회성 작업을 예약할 떄는 at을 사용한다.    

### 문제가 발생하거나 고민한 내용+ 해결 방법
* server b를 백업해놓지 않아서 처음부터 다시 설치했다.
